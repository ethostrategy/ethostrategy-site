Notes

Tradeoffs:
• Variant 0 (current) — most distinctive, feels alive. Costs a 60fps canvas loop on every page.
• Variant A — keeps the atmospheric gradient feel, no animation. Reads identically to 0 when the comet dots happen to be at the ends.
• Variant B — simplest, cleanest, most bulletproof at small sizes.
• Variant C — safest / most generic. Loses the brand mark.

To swap the main site's nav logo, change the <canvas id="logoCanvas"> element in each HTML file to the chosen SVG. Five minutes of find-and-replace.
